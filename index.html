PhantomSwift

The zero-dependency iOS debugging toolkit with 25 modules. Network inspector, memory leak tracker, 3D view hierarchy, and more. #if DEBUG safe.

View on GitHub Install Now

PhantomSwift is an open-source iOS debugging library for Swift developers. It provides network inspection, memory leak detection, UI hierarchy exploration, and 25+ diagnostic tools — all in a single zero-dependency package. Compatible with UIKit and SwiftUI, installable via SPM or CocoaPods. A modern alternative to FLEX, Netfox, and Pulse.

25 Debug Modules

Everything you need to debug, inspect, and optimize your iOS app — in one library.

🌐 Network Trace

Real-time HTTP/HTTPS traffic monitoring with full request/response inspection and HAR export.

🚧 Interceptor

Mock, block, delay, or redirect any request. Mockoon redirect support with hit counters.

📡 Bad Network

Simulate 3G, Edge, packet loss, and offline conditions with one tap.

⚡ Performance Monitor

Real-time CPU, FPS, and RAM tracking with interactive timeline graphs.

🧠 Memory Leak Tracker

Automatic retain cycle detection with object lifecycle tracking and visual memory graph.

🔍 3D View Hierarchy

Xcode-style exploded 3D view with tap-to-select, depth slider, wireframe mode, and zoom.

📝 Console Logger

Priority-level logging with tags, metadata, full-text search, and OSLog bridge.

💾 Storage Inspector

Browse and edit UserDefaults, Keychain, sandbox files, and SQLite databases.

🔒 Security Audit

Jailbreak detection, SSL pinning check, debugger detection, and binary integrity verification.

🚩 Feature Flags

Register, toggle, and persist feature flags at runtime with grouped UI.

📈 Analytics Inspector

Intercept and inspect analytics events from Firebase, Amplitude, and custom providers.

🖥 Remote Server

WebSocket debug server for real-time remote inspection from browser or Postman.

Installation

Add PhantomSwift to your Debug target only.

// Package.swift
dependencies: [
    .package(
        url: "https://github.com/synaptode/PhantomSwift.git",
        from: "1.0.0"
    )
]
# Podfile
pod 'PhantomSwift', :configurations => ['Debug']

Quick Start

Get up and running in under a minute.

import SwiftUI
#if DEBUG
import PhantomSwift
#endif

@main
struct MyApp: App {
    init() {
        #if DEBUG
        PhantomSwift.configure { config in
            config.triggers = [.shake, .dynamicIsland]
        }
        PhantomSwift.launch()
        #endif
    }
    var body: some Scene {
        WindowGroup { ContentView() }
    }
}
